# Phase 2 Task Pack

## Goal
Deliver trustworthy entity, identity, and skill review surfaces.

## Deliverables
- Teams and players directory/profile flows
- Identity graph/list fallback and linked-account management
- Consent registry and revocation UX
- Skill profile and merge suggestion review flows

## Implementation Tasks
1. Build entity profile templates with alias and roster sections.
2. Build identity graph with fallback list for degraded states.
3. Implement consent table with revocation safety copy.
4. Build skill confidence cards and merge review drawer.
5. Gate reviewer-only actions and owner-only self-service routes.

## Acceptance Gate
- Identity and merge actions are permission-safe, auditable, and confidence-explicit.
