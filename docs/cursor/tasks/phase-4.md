# Phase 4 Task Pack

## Goal
Bring governance, trust, and settings to operational parity.

## Deliverables
- Partner directory and grant matrix
- Approval and revocation flows with impact preview
- Trust signal queue, evidence panel, and disposition forms
- Audit and settings administration routes

## Implementation Tasks
1. Implement partner grant matrix with expiry and scope visibility.
2. Implement trust queue and evidence policy-safe redaction rules.
3. Implement audit table with actor/object/before-after surfaces.
4. Implement settings routes (members, roles, IdP placeholders, retention, notifications).
5. Add tests for revocation cascades and masked evidence behavior.

## Acceptance Gate
- Governance actions are explicit, auditable, and role-safe.
